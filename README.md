# 💰 Personal Finance Manager

A **full-featured MERN stack web application** to help users **track expenses, manage budgets, set financial goals, and plan their future**.  
Built with **MongoDB, Express, React, Node.js, TailwindCSS**, and **Chart.js**, it offers a complete and visually engaging personal finance management experience.

## 🧩 Tech Stack

   Layer                  Technology 
   **Frontend**        React (Vite), TailwindCSS, Axios, Chart.js 
   **Backend**         Node.js, Express.js, JWT Authentication 
   **Database**        MongoDB (Mongoose ORM) 
   **Other Tools**     pdfmake, json2csv, bcrypt, dotenv, cors 
   **Deployment**      Netlify (Frontend), Render (Backend) 



## 🌟 Features

### 🧾 Expense Tracking
- Record daily expenses with **amount, date, category, and description**
- Categorize expenses (Food, Rent, Utilities, etc.)
- Manage **recurring expenses** like subscriptions and rent

### 💸 Budget Management
- Create and manage **budgets** per category or time period (monthly/yearly)
- Real-time budget tracking with **alerts and notifications**
- Visual **budget vs. expense** comparisons

### 🎯 Financial Planning
- Set and track **financial goals** (e.g., savings, vacation fund)
- **Forecast future** expenses/savings using trend analysis
- Smart recommendations for budget optimization

### 📊 Financial Reports
- Generate **expense, budget, and income reports**
- **Visual dashboards** powered by Chart.js (pie, bar, and line charts)
- Export reports as **CSV or PDF** using json2csv & pdfmake

### 👤 User Profile & Settings
- Manage personal info and financial preferences
- Customize categories, currency symbol, and notification settings
- Secure authentication using **JWT tokens**

### 🔔 Notifications
- Alerts for approaching budget limits
- Reminders for **upcoming bills or recurring expenses**
- Progress updates for financial goals

### 💾 Data Export
- Export all data in **CSV**, **PDF**, or **ZIP** formats
- Download comprehensive financial summaries for offline use

### 📱 Responsive Design
- Fully responsive with **TailwindCSS**
- Optimized for **desktop, tablet, and mobile** devices
